## [mnk.gg](https://mnk.gg) (Mouse n' Keyboard)

Database to track which console games have mouse & keyboard support.

Built with [Svelte](https://svelte.dev/), [Supabase](https://supabase.com/) and hosted on [Vercel](https://vercel.com/).
